Diferenças entre div e span tag:

Tag HTML 'SPAN' o elemento HTML span é um contêiner embutido genérico para elementos e conteúdo embutido. É usado para agrupar elementos para fins de estilo (usando os atributos class ou id). A melhor maneira de usá-lo quando nenhum outro elemento semântico estiver disponível. A tag span é muito semelhante à tag div, mas div é uma tag de nível de bloco e span é uma tag embutida.

Tag HTML 'DIV': a tag div é conhecida como tag de divisão. A tag div é usada em HTML para fazer divisões de conteúdo na página da web como (texto, imagens, cabeçalho, rodapé, barra de navegação, etc). A tag div possui tags de abertura (div) e de fechamento (/div) e é obrigatório fechar a tag. Como sabemos, a tag Div é uma tag de nível de bloco. Neste exemplo, a tag div contém toda a largura. Ele será exibido tag div cada vez em uma nova linha, não na mesma linha.
